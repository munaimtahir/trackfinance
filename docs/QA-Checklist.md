# QA Checklist

- [ ] Father can log in without confusion.
- [ ] Father can:
  - [ ] Open the app and see a clear "Add New Bill" button.
  - [ ] Capture a bill photo.
  - [ ] Submit it and see a success message.
- [ ] Child can:
  - [ ] See new pending bills.
  - [ ] Open bill detail.
  - [ ] Mark bill as paid with a note.
  - [ ] Attach a receipt screenshot.
- [ ] Status updates:
  - [ ] Bills move from Pending to Paid list correctly.
- [ ] Notifications:
  - [ ] Child receives a notification when a bill is created.
  - [ ] Father receives a notification when bill is marked paid (if enabled).
- [ ] Offline:
  - [ ] If father is offline, app allows capturing a bill.
  - [ ] When back online, the bill syncs automatically.
